Methodology & Implementation

Cross-Validation Strategy

  • 5-Fold Cross-Validation: Used for all models to ensure robustness and prevent overfitting.
  • Stratified Sampling: Ensured class balance in training/testing splits where applicable.
  • Document-Level Splits: For Keyphrase Extraction, splits were done by document ID to prevent data leakage.

Tuning & Optimization

  • Keyphrase Extractor: Enhanced with TF-IDF features, positional binning, and 1:1 negative sampling.
  • Topic Model: Utilized Bigrams (e.g., "prime_minister") and asymmetric priors for better topic coherence.
  • Relation Extractor: leveraged deep syntactic dependency parsing via SpaCy.
